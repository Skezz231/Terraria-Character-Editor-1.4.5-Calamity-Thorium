# Support

If you need help with `Terraria 1.4.5+ Inventory & Character Studio`, use the most appropriate channel below.

## Usage questions

Open an issue if you need help with:
- building the project
- understanding features
- configuring the editor
- troubleshooting normal usage problems

## Bug reports

Please include:
- operating system
- Terraria version
- whether the save is modded or vanilla
- exact steps to reproduce
- expected result
- actual result
- screenshots if useful

## Feature requests

Please describe:
- the problem you want to solve
- the expected behavior
- why the feature is valuable

## Before opening an issue

- Check the `README.md`
- Check `CHANGELOG.md`
- Search existing issues first

## Save safety reminder

Always back up `.plr` files before editing.
